2.1 #Functions.brightness Brightness

Calculate Perceptual colour brightness. That is the brightness as users perceive them relative to each other. For example yellow (255,255,0) is mathematically the same brightness as magenta (255,0,255) but is perceived to be considerably lighter. Read more about the formula used.

Parameters:
  • $color
    The color to calculate the brightness of.
